# OBJECT: MessagePaginator

A paginated list of Message items.

## Estructura

| Campo         | Tipo             | Descripción                                     |
| :------------ | :--------------- | :---------------------------------------------- |
| paginatorInfo | `PaginatorInfo!` | Pagination information about the list of items. |
| data          | `[Message!]!`    | A list of Message items.                        |
