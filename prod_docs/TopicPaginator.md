# OBJECT: TopicPaginator

A paginated list of Topic items.

## Estructura

| Campo         | Tipo             | Descripción                                     |
| :------------ | :--------------- | :---------------------------------------------- |
| paginatorInfo | `PaginatorInfo!` | Pagination information about the list of items. |
| data          | `[Topic!]!`      | A list of Topic items.                          |
