# OBJECT: BundlePaginator

A paginated list of Bundle items.

## Estructura

| Campo         | Tipo             | Descripción                                     |
| :------------ | :--------------- | :---------------------------------------------- |
| paginatorInfo | `PaginatorInfo!` | Pagination information about the list of items. |
| data          | `[Bundle!]!`     | A list of Bundle items.                         |
