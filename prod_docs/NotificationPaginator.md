# OBJECT: NotificationPaginator

A paginated list of Notification items.

## Estructura

| Campo         | Tipo               | Descripción                                     |
| :------------ | :----------------- | :---------------------------------------------- |
| paginatorInfo | `PaginatorInfo!`   | Pagination information about the list of items. |
| data          | `[Notification!]!` | A list of Notification items.                   |
