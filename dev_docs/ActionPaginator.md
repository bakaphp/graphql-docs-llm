# OBJECT: ActionPaginator

A paginated list of Action items.

## Estructura

| Campo         | Tipo             | Descripción                                     |
| :------------ | :--------------- | :---------------------------------------------- |
| paginatorInfo | `PaginatorInfo!` | Pagination information about the list of items. |
| data          | `[Action!]!`     | A list of Action items.                         |
