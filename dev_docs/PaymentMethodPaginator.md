# OBJECT: PaymentMethodPaginator

A paginated list of PaymentMethod items.

## Estructura

| Campo         | Tipo                | Descripción                                     |
| :------------ | :------------------ | :---------------------------------------------- |
| paginatorInfo | `PaginatorInfo!`    | Pagination information about the list of items. |
| data          | `[PaymentMethod!]!` | A list of PaymentMethod items.                  |
