# OBJECT: ChannelPaginator

A paginated list of Channel items.

## Estructura

| Campo         | Tipo             | Descripción                                     |
| :------------ | :--------------- | :---------------------------------------------- |
| paginatorInfo | `PaginatorInfo!` | Pagination information about the list of items. |
| data          | `[Channel!]!`    | A list of Channel items.                        |
