# OBJECT: SocialChannelPaginator

A paginated list of SocialChannel items.

## Estructura

| Campo         | Tipo                | Descripción                                     |
| :------------ | :------------------ | :---------------------------------------------- |
| paginatorInfo | `PaginatorInfo!`    | Pagination information about the list of items. |
| data          | `[SocialChannel!]!` | A list of SocialChannel items.                  |
